Ce site web permet à un utilisateur de gérer son équipe de basketball.

Fenêtre principale :
Elle affiche les informations des matchs ainsi que quelques statistiques globales. De la même manière, il est possible de consulter les informations détaillées pour chaque joueur.

Feuille de match :
Pour chaque match qui n’a pas encore eu lieu, l’entraîneur peut créer une feuille de match. Il peut sélectionner 5 titulaires et jusqu’à 7 remplaçants maximum.

Après le match :
Une fois le match terminé, l’entraîneur peut saisir le résultat et le nombre de points marqués par chaque joueur.

Page statistiques :
Cette page permet de consulter des statistiques sur les joueurs, comme leur poste préféré et leur note moyenne attribuée par l’entraîneur.
